# Website Usage Tracker

## Description
A browser-based tool to track time spent on different websites and analyze productivity.

## Features
- Real-time website tracking
- Active tab monitoring
- Usage analytics dashboard
- Data storage using browser storage APIs

## Tech Stack
JavaScript, HTML, CSS, Chrome APIs

## How to Run
1. Clone the repository
2. Load extension in Chrome (Developer Mode)
3. Start tracking usage
